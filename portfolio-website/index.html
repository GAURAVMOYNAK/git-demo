HTML5

React.js

React.js is a JavaScript library developed by Facebook for building user interfaces, especially for single-page applications where components can be reused to manage complex UIs efficiently. It follows a declarative approach, making it easy to understand and maintain code.

CSS3

Vue.js

JavaScript ES6

Bootstrap

Bootstrap is a popular front-end framework that provides a set of pre-designed components and styles for building responsive and mobile-first websites. It includes a responsive grid system, typography, and a variety of components to streamline web development.

Python

Python is a versatile and high-level programming language known for its readability and simplicity. It is widely used for web development, data analysis, artificial intelligence, and more. Python emphasizes code readability and allows developers to express concepts in fewer lines of code.

Node.js

Next.js

Django

Django is a high-level Python web framework that encourages rapid development and follows the model-view-template (MVT) architectural pattern. With a clean, pragmatic design, Django aims to make it easier to build web applications by providing a robust set of tools for handling tasks such as database access, URL routing, and template rendering. It promotes code reusability, modularity, and follows the "Don't Repeat Yourself" (DRY) principle, making it an excellent choice for developers looking for efficiency and maintainability in their projects.

AngularJS

AngularJS is a JavaScript-based open-source front-end web application framework. Developed and maintained by Google, it extends HTML with additional attributes to make web applications more dynamic. AngularJS simplifies the process of building complex and single-page applications by providing a structured framework for developers to work with. It offers features like two-way data binding, dependency injection, and modular development, making it a powerful choice for building dynamic and interactive web applications.

jQuery

jQuery is a fast and lightweight JavaScript library that simplifies HTML document traversal and manipulation, event handling, and animation. It is designed to make things like HTML document traversal and manipulation, event handling, and animation much simpler with an easy-to-use API that works across a multitude of browsers.

Flask

Flask is a lightweight and easy-to-extend web framework for Python. It is designed to be simple and modular, allowing developers to choose the components they need. Flask is commonly used for building web applications and APIs.
